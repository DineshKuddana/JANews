Free Courses and Internships

Courses

HTML and CSS

Learn the basics of HTML and CSS with this free course from Codecademy: https://www.codecademy.com/learn/learn-html-css

Once you've learned the basics, you can improve your skills with this free intermediate course from FreeCodeCamp: https://www.freecodecamp.org/learn/responsive-web-design/

JavaScript

Learn JavaScript with this free course from Codecademy: https://www.codecademy.com/learn/introduction-to-javascript

Once you've learned the basics, you can improve your skills with this free intermediate course from FreeCodeCamp: https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/

Internships

Google Summer of Code

Google Summer of Code is an annual program where students work on open source software projects. Students receive a stipend for their work, and gain valuable experience and connections in the tech industry. Learn more and apply here: https://summerofcode.withgoogle.com/

Microsoft Explore

Microsoft Explore is a summer internship program for students who are interested in software engineering. Learn more and apply here: https://careers.microsoft.com/us/en/job/1122922/Explore-Internship-Opportunities
